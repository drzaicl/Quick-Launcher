# 🚀 Quick Launcher

A simple and powerful desktop application for launching multiple programs using custom profiles.

---

## 🧠 About

Quick Launcher is a Python GUI application that allows you to organize and launch your favorite applications with one click.

You can create multiple profiles (e.g. Work, Gaming, Custom), assign apps to each profile, and instantly run them whenever you need.

---

## ✨ Features

* 🔹 Multiple profiles (Work, Gaming, Custom)
* 🔹 Add / Delete / Rename profiles
* 🔹 Add / Remove applications per profile
* 🔹 Launch all apps in a profile with one click
* 🔹 Persistent data storage using JSON
* 🔹 Clean and simple dark UI

---


## ⚙️ How It Works

* Each profile contains its own list of applications
* Apps are stored as file paths
* Profiles and apps are saved in a local JSON file
* The app loads your data automatically on startup

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* JSON (data storage)
* OS module (launching apps)

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/yourusername/quick-launcher.git
```

2. Navigate to the project folder:

```
cd quick-launcher
```

3. Run the application:

```
python main.py
```
